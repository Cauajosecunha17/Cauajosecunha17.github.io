Tutoriais

Dinheiro

  • Mate mobs ofensivos para ganhar dinheiro.
  • Use /pay para transferir/pagar dinheiro para alguém.
  • Para adicionar dinheiro em espécie leia o tutorial do Banco.

Banco

  • Para adicionar itens como dinheiro vá até o Banco na cordenada (Em breve).
  • Consulte no Banco a lista de itens aceitos.
  • Deposite seus itens no baú do caixa.
  • Use /dbox e selecione a opção para escolher o baú.
  • Escolha o baú com seus itens depositados e confirme o depósito.

Clãns/Guildas

  • Para criar um clãn/guilda dê o comando /orgex my.
  • Após isso siga os passos no painel do plug-in.

Proteção de Terreno

  • Dê o comando /land new para mapear seu terreno.
  • Selecione a opção de marcação em 3D.
  • Vai no ponto A como mostra o exemplo e dê o comando /land set.
  • Agora vai no ponto B como mostra o exemplo e dê novamente o comando /land set.
  • Após isso aparecerá a janela para configurar a altura Y inicial e a altura Y final onde você vai colocar a altura mínima do seu terreno e a máxima onde suas extruturas alcançam.
  • Após isso dê o comando /land buy para comprar seu terreno.
